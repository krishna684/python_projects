# Python Projects

A collection of Python learning projects covering fundamental programming concepts through advanced applications.

## Project Overview

This repository contains 13 Python projects demonstrating:
- Input/output and string operations
- Control flow and loops
- Data structures (lists, dictionaries)
- Functions and modularity
- File I/O and data persistence
- Algorithms and mathematical operations
- Regular expressions and string processing
- Object-oriented principles
- Event-driven programming

## Projects

### Beginner Level
1. **[01_about_me](01_about_me/)** - Personal information display program
2. **[02_wage_calculator](02_wage_calculator/)** - Wage and train fare calculations
3. **[04_temperature_converter](04_temperature_converter/)** - Unit conversion utility

### Intermediate Level
4. **[05_leap_year_and_sum](05_leap_year_and_sum/)** - Leap year checker and sum calculator
5. **[06_math_operations](06_math_operations/)** - Factorial and odd number sum calculations
6. **[07_email_validation](07_email_validation/)** - Email format validation
7. **[08_list_operations](08_list_operations/)** - List manipulation and operations
8. **[09_email_domain_analyzer](09_email_domain_analyzer/)** - Domain frequency counter

### Advanced Level
9. **[10_shopping_calculator](10_shopping_calculator/)** - Price and shopping cost calculator
10. **[11_expense_tracker](11_expense_tracker/)** - Expense tracking system with persistence
11. **[12_caesar_cipher](12_caesar_cipher/)** - Encryption and decryption utility
12. **[13_budget_planner](13_budget_planner/)** - Budget calculation from expense data (Final Project)

## Getting Started

Each project folder contains:
- Python source code (.py files)
- Data files (if applicable)
- README.md with detailed documentation

### Running a Project

1. Navigate to the project folder:
```bash
cd 01_about_me
python aboutme.py
```

2. Follow the README in each folder for specific instructions and requirements

## Technology Stack

- **Language**: Python 3.x
- **No external dependencies** - All projects use only Python standard library
- **File formats**: .py (Python scripts), .txt (data files)

## Learning Outcomes

By completing these projects, you will:
- Master Python fundamentals and syntax
- Understand program flow and logic
- Work with data structures effectively
- Create modular, function-based code
- Handle file I/O operations
- Implement algorithms efficiently
- Build practical applications

## Project Progression

The projects are organized from simple to complex:
- **Projects 1-3**: Basic I/O, operations, and conversions
- **Projects 4-8**: Loops, conditionals, and data manipulation
- **Projects 9-12**: Complex logic, file handling, and algorithms
- **Project 13**: Comprehensive real-world application (final project)

## Notes

- Each project is self-contained with its own data files
- Projects use descriptive naming and modular function design
- Code includes error handling and input validation
- Suitable for beginners learning Python fundamentals
