# Wage Calculator

Calculates wages based on train schedules and hourly rates.

## Files
- `train.py` - Train schedule and fare calculations
- `wage.py` - Wage computation based on hours worked

## Description
These programs demonstrate arithmetic operations and conditional logic for:
- Computing transportation costs from train schedules
- Calculating wages from hourly rates and hours worked

## How to Run
```bash
python wage.py
python train.py
```

## Requirements
- User input for hours worked, hourly rate, or fare information
