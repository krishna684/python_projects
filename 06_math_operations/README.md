# Math Operations

Computes factorials and sums of odd numbers with looping and input validation.

## Files
- `factorial.py` - Factorial calculator
- `odd_total.py` - Sum of odd numbers calculator

## Description
### Factorial Calculator
- Takes a positive integer as input
- Computes factorial using a for loop
- Supports multiple calculations until user enters 'done'
- Validates input (must be positive)

### Odd Number Sum
- Accepts a positive integer as upper limit
- Sums all odd numbers from 1 to that limit
- Uses efficient looping patterns

## How to Run
```bash
python factorial.py
python odd_total.py
```

## Examples
- 5! = 120
- Odd numbers 1 to 10: 1 + 3 + 5 + 7 + 9 = 25
