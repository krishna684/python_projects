# Leap Year and Sum Calculator

Determines leap years and calculates sum of numbers with input validation.

## Files
- `leapyear_v2.py` - Leap year checker (version 2)
- `total.py` - Sum calculator

## Description
### Leap Year Checker
- Validates if a given year is a leap year
- Uses divisibility rules (4, 100, 400)
- Supports multiple year inputs until user terminates

### Sum Calculator
- Prompts user for numbers and calculates their sum
- Includes input validation
- Stops when user enters 'done'

## How to Run
```bash
python leapyear_v2.py
python total.py
```

## Leap Year Rules
- Divisible by 400 → Leap year
- Divisible by 100 → Not a leap year
- Divisible by 4 → Leap year
- Otherwise → Not a leap year
