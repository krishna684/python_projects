# Email Validation

Validates email addresses based on format requirements.

## Files
- `email.py` - Email validation logic

## Description
This program validates email addresses by checking:
- Presence of @ symbol
- Proper domain format
- Valid characters
- Minimum length requirements

## How to Run
```bash
python email.py
```

## Requirements
A valid email must have:
- Exactly one @ symbol
- Local part (before @)
- Domain name (after @)
- Top-level domain (after .)
