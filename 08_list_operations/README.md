# List Operations

Performs operations on Python lists including addition and manipulation.

## Files
- `addlists.py` - List addition and operations

## Description
This program demonstrates:
- Creating and initializing lists
- Adding/concatenating lists
- Accessing list elements
- Modifying list contents
- Iterating through lists

## How to Run
```bash
python addlists.py
```

## Operations Demonstrated
- List concatenation
- Element access and modification
- List iteration and manipulation
- Data type handling in lists
